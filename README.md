# Simple To-Do App

A simple and intuitive To-Do application built with React and Vite.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Responsive design with Tailwind CSS

## Demo

You can view the live demo of the application [here](https://golap12.github.io/Simple-To-Do-App/).
